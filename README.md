## ⑤工夫した点・こだわった点
 
URLは切り替えずに、同じページで　”表示のオンオフ” で見た目を変えているのですが、
リロードや戻る　を押した時に、同じ表示になるように、ローカルストレージに今の変数を記録して、
その記録に基づいて、firebaseに情報をとりに行くようにしました。
